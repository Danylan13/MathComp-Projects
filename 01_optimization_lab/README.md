# Optimization Lab

## Goal

Estimate apartment prices using a small synthetic dataset and compare two approaches:

- batch gradient descent
- closed-form least squares

## Why It Matters

This project demonstrates the practical side of numerical optimization: preprocessing, convergence tracking, and model evaluation.

## Concepts

- feature scaling
- linear regression
- mean squared error
- gradient descent
- normal equation

## Run

```bash
python 01_optimization_lab/src/optimization_lab.py
```

## Output

The script prints:

- learned weights
- loss evolution
- train and test RMSE
- a comparison between iterative optimization and the analytical solution
